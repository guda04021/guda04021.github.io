# My Name's Vinsent

“不想结婚的男人就该去死” 曾经是《凯瑟琳》单人故事当中一个定则，这个献祭仪式持续了至少数百年之久。
城郊的一隅，有一家叫做 "迷途羔羊" 的酒吧，店里没有禁烟席，除去披萨外没有什么正经的吃的，得益于店长不错的调酒水平，"迷途的羔羊" 还是吸引了不少客人。主角文森特就是其中的一只迷途羔羊。作为一名再就业的 system engineer, 工资不高不低，没有什么特殊技能，外貌平平无常，是个再不能普通的上班族.32 岁的他有一个交往了五年的女友凯瑟琳 (Katherine).
谈了这么久的恋爱，就是生米也该煮成粥了。凯瑟琳一直在暗示文森特 "到谈婚论嫁的时候了"

可文森特却被这突如其来的催婚而苦恼不已.
并不是不爱凯瑟琳，文森特只是觉得现在正是为工作而奋斗的大好时光，结婚这等大事应该等到事业有成之后在忙也不急，而且作为男人，他并不希望自己过早地被家庭所束缚，好不容易在成年后脱离父母的掌控，好不容易能在 30 岁有了部分的职业自由，好不容易... 不容易换来的自由他不希望过早地寄托在凯瑟琳的手中，他爱她，但他也希望凯瑟琳能给他想要的空间感；
疲惫了一天后，文森特来到可以吐露烦恼的地方 --"迷途的羔羊".
乔尼，和文森特一起长大的同学。成年后继承了老爹的修理厂，每天都带着小徒弟托比和哥们儿们一起喝酒。只想和还没遇到的 "命中注定的另一半" 结婚，和现在的女朋友没有之后的打算.
奥兰德，三人组当中唯一早早结婚的一个，做生意失败后和老婆离婚，现在过着潇洒的单身生活，自称 "婚姻就是一座坟墓，我就是现成的例子".
文森特将今天早上和凯瑟琳的谈话告诉了他们


他觉得自己不是讨厌凯瑟琳，只是畏惧婚姻.
可是，婚姻这种事情，三个大男人怎么能商讨出结果呢...
奥兰德将话题转移到了他的老同学，保罗身上.
"喂，你知道吗？保罗那家伙... 今天早上死了啊！"

事情发生地出奇就连警方也束手无策。人是在睡觉过程中死掉的，死状极其恐怖，全身如同缩水的纸片，弯曲狰狞.
正当众人为此诧异时，文森特的手机响了.

又是要自己赶紧结婚
...
...
文森特已经无心继续在酒吧呆下去了.
